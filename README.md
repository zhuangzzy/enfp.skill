## 🚀 全平台使用方法
### 1. 任意AI工具直接调用
复制`SKILL.md`+`prompt.txt`内容，粘贴到Claude/GPT/豆包，直接触发技能

### 2. 开源Agent平台导入
- **OpenClaw/QCLAW**：执行`skill install https://github.com/zhuangzzy/enfp.skill`一键安装，`@enfp`直接调用
- **Dify**：导入技能文件，封装为独立应用，导出API/多平台机器人

### 3. 私有化部署
克隆仓库到本地，集成到个人AI助手，数据100%可控
